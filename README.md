# AI-Based Sentiment Analyzer

An **AI-powered Sentiment Analysis web application** that analyzes user text (reviews, comments, feedback) and classifies it into **Positive, Neutral, or Negative** sentiment using **Hugging Face Transformers**.  
The project features a clean and modern UI with a **Dark Lavender Gradient** background.
